# 🌟 Overview

**AI Database Assistant** is an intelligent application that allows users to interact with SQL databases using **natural language** instead of writing SQL queries manually.

Simply ask questions like:

> *"Show all customers who placed orders last month."*

and the assistant automatically:

* 🧠 Understands your request
* ⚡ Generates optimized SQL
* 🗄️ Executes the query
* 📊 Returns formatted results instantly

It combines the power of **Large Language Models** with traditional databases to create a smarter querying experience.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Samar-111/AI-database-assistant.git

cd AI-database-assistant
```

---

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn app:app --reload
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
